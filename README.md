# SLEEPYLYCHEE — Art

Visual & generative art tools by SLEEPYLYCHEE.

## Tools

### [BLOOM](https://lychee23mxp.github.io/sleepylychee-art/bloom/) — Gradient Flower Generator
Interactive animated flower tool with BPM-driven rotation.

- **50–250 BPM** controls rotation speed, synced to beat pulse
- **Tap Tempo** button for live BPM setting
- Adjustable petals, size, petal width, layers
- 3-stop gradient (inner / mid / outer) with slow color drift animation
- Glow, trail/persistence, beat pulse intensity
- CW / CCW rotation toggle
- Save PNG
